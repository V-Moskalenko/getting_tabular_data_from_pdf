Добро пожаловать в документацию сервиса "Получение табличных данных из PDF"
=============================================================

Данный сервис или RPA, позволяет распознавать табличные данные из PDF-файлов, с
помощью Tesseract-OCR и декстопным приложением FineReader.
Для удобства использования функционал внесен в Telegram бот.
