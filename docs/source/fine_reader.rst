Модуль работы с FineReader
***********************

Расположение
############
sources -> fine_reader_module -> pywinauto_fr.py

Описание
########
Модуль работы с десктопным приложением ABBYY FineReader.
Проводит конвертацию принятого от пользователя PDF-файла в excel-формат,
таким образом мы получаем один из возможных вариантов распознаных данных.

Принцип работы модуля
#####################
   1. Открывает установленное приложение ABBYY FineReader по указанному пути до FineReader.exe файла
   2. Ждёт готовности загрузки главного окна приложения
   3. Нажимает на кнопку "Конвертировать в Microsoft Excel"
   4. Переключается на окно выброва файла для конвертации и указывает путь до PDF-файла
   5. Нажимает на кнопку запуска конвертации "Конвертировать в Excel"
   6. Вводит путь сохранения конечного excel-файла
   7. Снимает checkbox "Открыть документ после конвертации", если он был установлен
   8. Ожидает конвертации файла
   9. Закрывает приложение


Конфигурация
############
Для работы модуля, необходимы следующие настройки:
   1. Путь, до запуска программы ABBYY FineReader -> "FineReader.exe"
   2. Путь до PDF-файла, который необходимо распознать (первый аргумент функции)
   3. Путь до распознанного excel-файла, для его создания (второй аргумент функции)
