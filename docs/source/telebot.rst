Модуль работы с pyTelegramBotAPI
***********************

Расположение
############
sources -> telegram_bot -> telebot_module.py

Описание
########
Синхронный Telebot для простого взаимодействия пользователей с данным сервисом.
Функционал реализован на кнопках и предпологает 3 шага взаимодействия:
   1. Выбор кнопки распознавания
   2. Выбор кнопки наклона изображения
   3. Загрузка PDF-файла
Подробная иснструкция указана в стартовом сообщении бота.

Принцип работы модуля
#####################
   1. Вывод стартового сообщения с инструкцией
   2. Получение текста от нажатой кнопки "Получить данные со сканированного изображения" и его проверка
   3. Получение наклона изображения файла
   4. Получение от пользователя PDF-файла
   5. Загрузка файла
   6. Запуск основной функции распознавания recognition_main
   7. Возврат пользователю результата распознавания в excel-файле
   8. Удаление файлов из системы

Конфигурация
############
Для работы модуля, необходимы следующие настройки:
   1. Токен Telegram бота от BotFather
   2. Путь до Poppler, библиотеки конвектирующей pdf в изображение, находится в папке utils проекта
   3. Путь до PDF-файла, который необходимо распознать (первый аргумент функции)
   4. Наклон изображения, число символизирующее градусы (второй аргумент функции)
